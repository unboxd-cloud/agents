# Unboxd Cloud Business Model

## Category

**Agent-Native Cloud Infrastructure & Platform Engineering**

Unboxd Cloud creates value by packaging open-source, cloud-native, Kubernetes-mature infrastructure into usable, governed, production-ready managed solutions.

## Revenue Model

```text

+
Usage Revenue
+
Managed Operations Revenue
+

+

+
```

### Setup Revenue

One-time revenue for getting the customer live.

Examples:

- Assessment
- Architecture
- Deployment
- Migration
- Integration
- Configuration
- Training

Examples:

- Compute
- Storage
- Bandwidth
- Apps
- Environments
- Agent runs
- Workflow executions
- Observability retention
- Backup storage

### Managed Operations Revenue

Recurring revenue for running the system.

Examples:

- Monitoring
- Patching
- Backups
- Upgrades
- Incident response
- Cost review
- Security checks
- Monthly improvement

### Support Revenue

Recurring revenue for human support.

Examples:

- Standard support
- Priority support
- SLA support
- Dedicated support
- Enterprise escalation

### Governance Revenue

Premium revenue for trust, risk, and control.

Examples:

- Identity governance
- Policy governance
- Agent governance
- Audit reports

- Compliance evidence

### Enterprise Revenue

High-value custom contracts.

Examples:

- Regulated workloads
- Custom integrations
- Co-managed operations

## Unit Economics

```text
Revenue per customer
-
Infrastructure cost
-
Support cost
-
Operations cost
-
Delivery cost
=
Gross margin
```

Cost drivers:

- Compute
- Storage
- Bandwidth
- Backups
- Monitoring retention
- Support hours
- Incident response
- Custom integrations
- Governance reviews

High-margin areas:

- Templates
- Automation
- Usage-based infrastructure
- Self-serve onboarding
- Standard managed operations

Lower-margin areas:

- Custom implementation
- Manual support
- Complex integrations
- Enterprise governance
- Air-gapped deployments

Business sustainability depends on reusable open-source platform components reducing delivery cost, usage-based pricing scaling revenue with consumption, and managed operations/support creating recurring margins.

## Final Business Model Definition

> Unboxd Cloud is an open-source, usage-based, agent-native cloud infrastructure business that creates value through reusable platform primitives and captures value through solution delivery, managed operations, enterprise support, governance services, and usage-based pricing.

Compact version:

```text
Open Source for adoption.
Usage-based pricing for scale.
Managed operations for recurring revenue.
Governance for enterprise value.
```
